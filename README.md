# Dependency Injection using Intravenous for Node.js Application

## Installation

This will install Intravenous from npm repository.
```bash
npm install
```

## Run

```bash
npm src/app
```

## Notes

- Pros
    - Able to dispose.
- Cons
    - Need to explicitly register all the dependenciens in advance.
